---
title: Algoritmos
---

Olá, jovens exploradores! Hoje, vamos embarcar em uma aventura emocionante no mundo dos algoritmos. Mas o que é um
algoritmo? Bem, um algoritmo é como uma receita. É uma série de passos que você segue para fazer algo. No nosso caso, é
o computador que segue os passos para resolver um problema ou realizar uma tarefa.

**Algoritmo** é um passo a passo para chegar a um objetivo final.

Algoritmos são como uma receita de bolo, ou as instruções sequenciais para
chegar à padaria mais próxima da sua casa:

1. Abra a porta
2. Atravesse a porta
3. Feche a porta
4. Caminhe em direção ao elevador
5. Pressione o botão para chamar o elevador
6. Aguarde até o elevador chegar
7. Entre no elevador
8. Pressione o botão do térreo
9. Aguarde até o elevador chegar
10. Saia do elevador
11. Vá até o portão do prédio
12. Abra o portão
13. Atravesse o portão
14. Feche o portão
15. Caminhe por 200 metros
16. Vire à esquerda
17. Caminhe por 50 metros
18. Vire à direita
19. Caminhe por 20 metros

Programar é escrever algoritmos, ou seja, escrever instruções que um computador
precisa seguir para alcançar um objetivo final.
