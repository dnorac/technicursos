Bem-vindo de volta, explorador do código! Agora que sabemos o que são algoritmos, precisamos aprender a **guardar informações** durante nossa jornada. Assim como um aventureiro carrega uma mochila cheia de itens, os programas precisam armazenar dados em **variáveis** e **estruturas de dados**.

### **1. O que são Variáveis?**

Imagine que você está em uma caça ao tesouro e precisa lembrar quantas chaves encontrou pelo caminho. Para isso, usamos uma **variável**, que funciona como uma caixa onde guardamos valores.

Exemplo:

```java
int chaves = 3;
System.out.println("Você tem " + chaves + " chaves do tesouro!");
```

Aqui, criamos uma **variável chamada `chaves`** e guardamos o número 3 nela. Quando queremos usar essa informação, basta chamá-la!

---

### **2. Tipos de Variáveis**

Cada item na sua mochila pode ter um tipo diferente. No código, também usamos tipos de variáveis para armazenar diferentes tipos de informações:

- **int** → Números inteiros (ex: 10, -5, 42)
- **double** → Números com decimais (ex: 3.14, 9.99)
- **char** → Letras (ex: ‘A’, ‘B’, ‘C’)
- **boolean** → Verdadeiro ou falso (ex: `true`, `false`)
- **String** → Texto (ex: "Tesouro encontrado!")

Exemplo:

```java
String tesouro = "Espada de Ouro";
System.out.println("Você encontrou: " + tesouro);
```

---

### **3. Estruturas de Dados - Guardando Vários Itens**

Se um aventureiro precisa carregar **várias chaves**, ele não pode criar uma variável para cada uma. Para isso, usamos **arrays e listas**, que guardam múltiplos valores.

Exemplo de um **array**:

```java
String[] mochila = {"Poção", "Mapa", "Lanterna"};
System.out.println("Itens na mochila: " + mochila[0] + ", " + mochila[1] + " e " + mochila[2]);
```

Aqui, criamos uma **mochila** com três itens e acessamos cada um deles pelo índice.
