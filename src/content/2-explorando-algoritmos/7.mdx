**O Desafio: Criando o Algoritmo dos Tesouros!**

Agora que você tem as ferramentas e o conhecimento para organizar e manipular os tesouros, seu objetivo é criar um algoritmo que resolva um problema complexo. Que tal um algoritmo que organiza os tesouros de uma maneira específica e ainda pode ser usado para resolver desafios no mundo real?

A missão é criar um algoritmo para organizar os tesouros de forma que você consiga, com eficiência, encontrar os maiores ou menores tesouros em um grande conjunto, tudo de uma maneira otimizada!

### **Como Criar um Algoritmo?**

Criar um algoritmo é como traçar um mapa para uma nova aventura. Você precisa planejar os passos que o algoritmo vai dar, para que ele consiga resolver o problema de forma rápida e correta. Vamos ver o passo a passo para construir um algoritmo eficiente:

1. **Entenda o Problema**: Antes de tudo, é essencial entender o que o problema pede. Qual é o seu objetivo? Que tipo de tesouro você precisa organizar? Por exemplo: você quer organizar os tesouros por ordem de valor? Ou deseja procurar o maior tesouro? 🧐

2. **Divida o Problema em Partes Menores**: Assim como nas listas encadeadas, o segredo está em dividir o trabalho em partes menores. Você pode criar um algoritmo que primeiro organiza os tesouros em partes, depois ordena essas partes, ou ainda que procure o maior ou menor tesouro em uma sequência.

3. **Escolha o Método Certa**: Agora, use tudo o que aprendeu sobre os métodos de ordenação (Bubble Sort, Merge Sort, Quick Sort) ou até mesmo listas encadeadas para resolver o problema. Qual será a maneira mais eficiente de organizar os tesouros? 🤔

4. **Escreva o Algoritmo**: Com a estratégia definida, agora é hora de escrever o algoritmo. Lembre-se de que cada passo do algoritmo deve ser claro, lógico e eficiente!

### **Exemplo: Algoritmo de Ordenação e Busca de Tesouros**

Vamos criar um algoritmo simples para organizar e depois procurar pelo maior tesouro de um conjunto de gemas! Imagine que você tem os seguintes valores de tesouros (gemas):
**[5, 2, 9, 1, 6]**

Aqui está o algoritmo em passos simples:

**Passo 1: Organize os Tesouros**

1. Pegue os tesouros e organize-os de forma crescente usando o _Bubble Sort_.

   Resultado após o _Bubble Sort_:
   **[1, 2, 5, 6, 9]**

**Passo 2: Encontre o Maior Tesouro**

1. Depois de organizar os tesouros, percorra a lista e pegue o maior valor (último tesouro na lista organizada).

   Maior Tesouro: **9**

Esse é um exemplo simples de como você pode usar os conceitos de algoritmos de ordenação para resolver problemas. Agora, a missão é criar algo ainda mais interessante!

### **Desafio: Crie Seu Próprio Algoritmo!**

Aqui está o seu desafio final, aventureiro! Use os conceitos de algoritmos, listas encadeadas, e ordenação que você aprendeu para resolver o seguinte problema:

**Problema:**
Você tem uma lista de tesouros com valores misturados. Crie um algoritmo que:

1. Organize os tesouros.
2. Encontre o menor tesouro.
3. Encontre o maior tesouro.

**Dica:** Use o método de ordenação que achar mais interessante ou crie seu próprio algoritmo! Não se esqueça de testar o seu algoritmo com diferentes listas de tesouros!
